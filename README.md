# LiFi-System
Li-Fi Communication System

Developed a Li-Fi (Light Fidelity) based communication system for wireless data transmission using visible light communication technology.

Project Overview

The project demonstrates data transmission through LED light signals instead of traditional radio frequency communication methods.

Features

- Wireless optical communication
- LED-based data transmission
- High-speed light fidelity concept
- Basic receiver and transmitter implementation

Tools & Technologies

- Embedded Systems
- Electronics Components
- Signal Processing Concepts
- Proteus Simulation

Learning Outcomes

- Understanding of visible light communication
- Practical exposure to wireless communication concepts
- Hardware interfacing and signal transmission basics

Status

Academic minor project completed during undergraduate coursework.
